อัปเดต:  |  แหล่งข้อมูล: Embedded
📋 วิธีเชื่อมต่อ Google Sheets (ต้องทำครั้งแรกครั้งเดียว)
① เปิด Google Sheets
เปิดไฟล์ แผนเก็บเงิน คอนโด ใน Google Sheets
② File → Share → Publish to the web
เมนูบาร์ → File → Share → Publish to the web
⚠️ ไม่ใช่ "Share" ปกติ — ต้องเป็น "Publish to the web"
③ เลือก Sheet + Format
ช่องซ้าย: เลือก Transaction1 (หรือ Entire Document)
ช่องขวา: เลือก Comma-separated values (.csv)
④ กด Publish แล้วกลับมากด
กด Publish ใน popup → กลับมาที่ Dashboard → กดปุ่ม 🔄 โหลดจาก Google Sheets
⚠️ "Anyone with the link" ≠ Publish to the web — การ Share แบบปกติยังต้องล็อกอิน Google ถึงเข้าได้ การ Publish to the web เท่านั้นที่ให้ fetch แบบไม่ต้องล็อกอินได้
🚀 วิธีเชื่อมต่อ Google Sheets (แนะนำ)
เปิด Terminal แล้วรัน:
cd "ที่อยู่ไฟล์" && python3 -m http.server 8080
จากนั้นเปิด: http://localhost:8080/condo_dashboard.html
หรือดับเบิลคลิกไฟล์ เปิด Dashboard.command ในโฟลเดอร์เดียวกัน (macOS)
